+-+-+-+-+-+-+-+
|  ゲーム概要 |
+-+-+-+-+-+-+-+
閉じ込められたアミューズメント施設から脱出するゲームです。
電力を復旧した後施設内のゲームをプレイしパスワードを入手し
出口のドアのロックを解除出来たらクリアとなります。
しかしその間にピエロがプレイヤーを狙い追いかけてきます。
捕まってしまえばゲームオーバーとなってしまいます。

//===========================
// プロトタイプ版
//===========================
プロトタイプ版は、ステージ配置とミニゲームの実装をメインに制作しました。
現状は出口に触れるとゲームクリア判定となります。
敵に触れたらゲームオーバーとなります。
ミニゲームはUFOキャッチャー、シューティングがプレイ可能です。
UFOキャッチャーは白い筐体、シューティングは青い筐体に近づくとプレイ可能です。
Cを押すと開けるマップを参考に探してみてください。
