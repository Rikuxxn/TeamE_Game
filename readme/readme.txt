+-+-+-+-+-+-+-+
|  ゲーム概要 |
+-+-+-+-+-+-+-+
閉じ込められたアミューズメント施設から脱出するゲームです。
電力を復旧した後施設内のゲームをプレイしパスワードを入手し
出口のドアのロックを解除出来たらクリアとなります。
しかしその間にピエロがプレイヤーを狙い追いかけてきます。
捕まってしまえばゲームオーバーとなってしまいます。
